Results List Formatter

The results list formatter is used to rearrange the order of the columns in the results list.

Press the up button to move the selected column up in the list [further to the left in the results list]

Press the down button to move the selected column down in the list [further to the right in the results list]

Press the OK button to save your new format and redisplay the results list

Press the Cancel button to discard any changes and return to the main screen
